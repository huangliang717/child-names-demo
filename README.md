# 儿童识字小报生成器

一款基于 AI 的儿童识字小报生成工具，通过 Nano Banana Pro API 自动生成精美的识字小报图片。

## 功能特点

- 🎨 **智能词汇联想**：根据场景自动推荐 15-20 个相关词汇
- 📝 **模板化提示词**：基于专业提示词模板生成绘图指令
- 🔒 **本地存储**：API Key 保存在浏览器本地，安全可靠
- 📱 **响应式设计**：支持手机和电脑使用

## 快速开始

### 1. 获取 API Key

1. 访问 [Kie AI API Key 管理](https://kie.ai/api-key)
2. 注册账号并获取 API Key
3. 首次使用时在页面输入框中粘贴 API Key

### 2. 使用步骤

1. 打开 `index.html` 文件（双击或在浏览器中打开）
2. 选择场景（超市、医院、公园、学校等）
3. 输入小报标题，如《走进超市》
4. 点击「开始生成识字小报」按钮
5. 等待 AI 生成完成后，下载图片
